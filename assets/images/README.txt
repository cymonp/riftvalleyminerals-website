Notes
-----
- Filenames are case-sensitive on many web hosts. Match exactly.
- Recommended: compress JPGs to ~200 KB each before uploading so
  the site loads fast. Tools: tinyjpg.com, squoosh.app.
- PNG for the logo is fine; keep it transparent-background if the
  original is transparent.
